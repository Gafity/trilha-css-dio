Seletores Css

Bem-vendo!

No Css nós possuimos os seletores por ID

  • Seletor por tipo:Busca elementos por uma tag HTML especifica
  • Seletor por ID (#):Busca elementos atraves do atributo "id"
  • Seletor por Classe (.):Busca elementos atraves do atributo "class"
  • Agora com duas class(.):
  • Seletor por Universal (*):Seleciona todos os elementos HTML
  • Seletor por Atributo ([atrib]):Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
  • Exemplo de seletor por atributo Netflix

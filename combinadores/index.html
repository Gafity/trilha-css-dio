Agrupamento de Seletores

Podemos aplicar a mesma regra CSS para vários seletores diferentes.

Para isso, podemos separar os seletores utilizando uma viírgula (,)

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS reutilizar as mesmas regras!
